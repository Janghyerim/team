오라클 SQL git 올리기 

git bash 드라이브 변경 :  cd e:  

git 파일 생성 : git init

ls : 파일 목록 보기 

git status :  올릴 목록 보기

git add ' name '   : 하나의 폴더 업로드
git add .      : 해당 폴더에 있는 모든 폴더 업로드 

git commit -m " mes"  : 커밋 메시지 작성 

git remote add origin 올릴 본인 레파지토리 주소 : 레파지토리와 연결 

git remote -v  : 연결 확인 

git push origin master  : 최종 업로드 


git remote rm origin : 레파지토리 연결 해제
